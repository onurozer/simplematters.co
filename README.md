# Simple Matters

Source code for [simplematters.co](http://simplematters.co)

## Requirements

* Ruby v3.3.0
* Node v20.11.1

## Installation

Run bundle and yarn

```bash
bundle install
yarn install
```

## Usage

Run the server and visit [http://localhost:4000](http://localhost:4000)

```bash
bundle exec jekyll server --livereload
```
